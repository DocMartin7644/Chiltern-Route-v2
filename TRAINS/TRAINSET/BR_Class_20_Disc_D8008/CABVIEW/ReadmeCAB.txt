Cabview for UK Class20 Diesel locomotive.
This is nose-front type cabview.

-------------------------------------------------------------
Install:
 1 Backup and delete the oroginal folder of [CABVIEW].
 2 Unzip [20cabRV1.zip] and put the folder of [CABVIEW]
   into your trainset(ex.[BATS_Class20_Green]...).
 3 Change next line of eng-file of D8003.

   CabView ( xxxx.cvf ) ----->  CabView ( class20R.cvf )

If you use common.cab;
 1 Backup and delete the oroginal folder of [CABVIEW].
 2 Unzip [20cabRV1.zip] and rename the folder of [CABVIEW] to [20cab].
 3 Put [20cab] into the folder of [common.cab].
 3 Change next line of eng-file of D8003.

   CabView ( xxxx.cvf ) ----->  CabView ( class20RC.cvf )


And, if you use Green color class20;
 1 Change next line of eng-file of D8003;

   CabView ( xxxx.cvf ) ----->  CabView ( class20RG.cvf )

 2 If you use common.cab;

   Change next line of eng-file of D8003.

   CabView ( xxxx.cvf ) ----->  CabView ( class20RGC.cvf )
