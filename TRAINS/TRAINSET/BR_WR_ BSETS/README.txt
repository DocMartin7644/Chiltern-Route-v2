GWR Coaches Mukltipack
++++++++++++++++++++++

A multipack of GWR Coaches in BR livery.  Included are a set of BSet non corridor coaches in BR Crimson and Maroon and a toplight.

All original models by Robin Howell.

Some textural ajustments and packaging by David Ward

To install, simply run the easy to use self installer programme.
