exGWR Diagram V37 Covered Ply Van.
V1.0

3D Model built from scratch by Robin Howell (robin.howell@tiscali.co.uk)
repainted by Ian Macmillan
-----------------------------------------------------------------------------
The 3D model has been built using TrainSim Modeler
Built 1947 to diagram V37. Faced with Plywood rather than built up by wooden planks due to wartime timber shortages.


The wagons have been built specifically for use on the Highworth branch. See www.highworth.freeuk.com 

Any suggestions of help or constructive comments welcome.
