GW  DiagramO18 16foot 5Plank Open.
2475 wagons were built at Swindon between 1914 and 1924. Originally fitted with Dean-Churchward brake gear, later builds had Morton brakes.
Both types are modelled.

GW  DiagramO24 17ft 6in 5Plank Open.
1815 wagons were built at Swindon between 1923 and 1929.

GW  DiagramO31 17ft 6in 5Plank Open.
150 wagons were built at Swindon between 1932 and 1933.
Fitted with vacuum brakes.

They lasted in traffic until the mid 60s but many then passed to the engineers as spoil carriers.

These wagons were previously exclusive to the Bala route CD.
Version 3 upgrades the underframe detail and adds additional loads.
Also added is a version of the 018 wagon with GWR DC brakes.
If you have the previous version of this wagon, allow to overwrite if asked.


INSTALLATION
~~~~~~~~~~~~

Unzip to a temporary folder.
Click on the exe file and the exGW_5Plank_opens and loads will be installed your Trainset folder

You are now ready to create a consist containing this model by using the Activity Editor supplied with Microsoft Train Simulator - the model will not be available in the simulator unless you do this.

These wagons require Stuart Williamson's  SWB Wagon Sounds  File ID 3290
