Great Western Railway Class 5101 " Large Prairie "
Version 1.0 ( July 2007 )

3D Model built by Reg Furniss

-----------------------------------------------------------------------------
The 3D models have been built using TrainSim Modeler

The Great Western Railway ( GWR ) 5101 Class of Large Prairie 2-6-2T locomotive was
designed for stopping passenger work, and were found all over the GWR system.
Some 10 examples have been preserved and included in this file are
No.5164 in Great Western livery as on Seven Valley Railway
No.5199 in BR livery as on The Llangollen Railway
You will need Stuart Williamson`s " 2 cylinder generic sound set " UKTS File ID2910
( add it to your common.sound folder )
You will need Lew Price`s " Praire Tank " CabView file UKTS ID 8519 ( add it to 
your common cab folder )
