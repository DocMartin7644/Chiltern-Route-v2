Great Central  Brake Vans.

Version: 1.0
Released : 24/04/2018
Models and textures : Ian Macmillan


PROTOTYPE
~~~~~~~~~~
The first brakevan built in large numbers was a 15 ton 10 ft wheelbase single balcony design to diagram  71
They lasted until WW2.
They were followed in 1906 by a 6 wheel double balcony van to diagram 73 which lasted into the mid 1950s.

Similar vans were built for the Cheshire Lines Committee

THE MODELS
~~~~~~~~~~
I have included vestibule leading and vestibule trailing versions of the single end vans.
You should not filp this model in the activity editor as this flips the lightglow but not the freight animation lamps..

 
INSTALLATION
~~~~~~~~~~~~

Unzip to temporary folder and Copy, Drag or Move the 3 folders to your Trainset folder. 

You are now ready to create a consist containing this model by using the Activity Editor supplied with Microsoft Train Simulator - the model will not be available in the simulator unless you do this.
WARNING: It is not recommended that you delete these files once installed without deleting the activity and consist files that use them.
