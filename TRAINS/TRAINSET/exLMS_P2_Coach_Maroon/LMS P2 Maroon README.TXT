Installation
~~~~~~~~~~~~
An auto installer is included that will install these coaches into the default MSTS location on your computer.  If you wish to unzip the files to another location or if you multiple MSTS installations then you will need to specify this when prompted.

UKSteamTrack - Part 1 Bogie Coaches Enhanced Sound is required.  

WARNING: It is not recommended that you delete these files once installed.
